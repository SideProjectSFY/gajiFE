# Shared Components (Park UI)

Use this directory for shared UI components built with PandaCSS + Park UI patterns.

Examples to place here:

1. `Button.tsx`
2. `Dialog.tsx`
3. `Input.tsx`
4. `Card.tsx`

Guideline:

1. Keep domain-agnostic UI in `src/components`.
2. Keep feature-specific UI in `src/domains/*/components`.
